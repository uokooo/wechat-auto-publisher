- 19 种排版风格：默认 / 晚点 / FT / Claude / 技术 / NYT / Jony Ive / Apple 极简 / 原研哉·空 / 高迪·有机 / Guardian / Nikkei / 焦橙 / Le Monde / Stripe / 橙 / 绛/绯/抹茶
