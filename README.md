# 📸 自动发布公众号文章

> *只想发图不想码字？交给 AI 吧。*

出去旅游拍了照片，懒得写文章？把图丢过来，AI 自动识别内容、配文配图、排版美化，三秒搞定一篇公众号文章。在电脑前？打开网页版排版器，19 种风格随便切，一键复制到公众号。

---

## ✨ 它能做什么

### 🎯 场景一：人在外面，只有手机

```
你：「发公众号」+ 发几张照片
AI：自动识别照片内容 + 写文章 + 推送到草稿箱
你：打开手机微信 app，点一下「发布」
```

### 🎯 场景二：坐在电脑前，想好好排版

```
AI：文章写好了
你：选②美化
AI：Markdown 写入本地排版器，发链接给你
你：浏览器打开 → 切风格 → 复制 → Ctrl+V 到公众号
```

### 🎯 场景三：还没想好发不发

```
选③存档：AI 帮你存 HTML 归档
过几天想发了：调入排版器 / 直接 MD 发表，都行
```

## 🚀 快速开始

### 方式一：AI 助手全自动（推荐）

找一个人工智能助手（需要支持工具调用），对着它说 **"发公众号"**，然后发照片就行。它会自动完成：识图 → 写文 → 排版 → 发布三选一。

### 方式二：纯手动使用排版器

1. 下载 `index.html`，浏览器打开
2. 左侧粘贴 Markdown 内容
3. 点风格按钮切换主题
4. 点「复制到公众号」
5. 到微信公众号编辑器按 Ctrl+V
